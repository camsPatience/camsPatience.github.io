About this project
This is a recreation of the Camp Menu from the Kingdom Hearts series, largely taking after Kingdom Hearts II. It is free to use, so long as you credit this Github repository.
If an entry has a star/ability icon on it, you can hover that icon to view the note for that entry.
(This project is designed for desktop PCs that display at 1280x720 or higher. Please enable audio auto-play for sound effects.)
